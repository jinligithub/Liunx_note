一.指令

1.ls指令
语法：ls  [选项] [目录或文件]

功能：
1）对于文件，列出文件名以及其他信息。
2）对于目录，列出该文件下的所有子目录与文件。

常用选项：
-a ：列出文件下的所有文件，包括隐藏文件。
-d：(ls -d：列出指定文件)
-i：输出文件的i节点（inode:文件的inode编号，一个文件对应的有一个inode）信息。 ls -ai :指定文件
-k：以K字节的形式表示文件的大小，ls  -alk 指定文件
-l：列出文件的详细信息。显示文件的各种属性，权限，大小，时间等
-n：用数字的UID，GID代替名称。
-F：在每个文件名后附上一个字符以说明该文件的类型（“*”表示普通文件；“/”表示是目录；“@”表示是连接；“I”表示FIFOs；“=”表示套接字（sockets））

注意事项
1.".文件名"表示隐藏文件
2.“.”表示当前目录
3.“..”表示上一级目录
4.“/”表示根目录（路径中间的/表示分隔符）（“/”表示根目录因为文件都是从根目录开始的）
5.相对路径：它不是从根目录开始的，所以他不是唯一的
6.绝对路径：它是从根目录开始向后访问目录的路径，它是唯一的

3.cd指令（change  dir）
在Linux下，磁盘中的文件和目录组成了一颗目录树，每个节点都是目录或者文件。
语法：cd  目录名
功能：改变工作目录
常用选项
cd  .. ：返回上一级目录
cd /home/......：绝对路径（唯一的，从根目录开始）
cd .../file/：相对路径（不唯一）
cd  - ：返回最近访问的目录
cd  ~：进入用户的家目录（当前用户的主工作目录）

主工作目录：Windows就是我的文档；Linux下就是刚打开Linux后直接pwd,显示出来的就是主工作目录

5.mkdir指令
语法：mkdir [选项] 目录名
功能：在当前目录下创一个目录
常用选项：
-p ,-parents  可以是一个路径名称，此时若路径中的某些目录不存在，加上此选项后，系统将建立好那些不存在的目录，即一次可以创建多个目录。

6.cp指令
语法：cp  [选项] 源文件或目录   目标文件或目录
功能：复制文件或目录
 说明：cp命令用于复制文件或者目录，
1）如同时指定两个以上的文件或目录，且最后得目的地是一个已经存在的目录，则会把前面指定的文件或目录复制到此文件中 ，
2）若同时指定了多个文件或目录，而最后得目的地并非一个已经存在的目录，则会出现错误。
常用选项
-f 或-force：强行复制文件或目录，不论文件或目录是否已经存在
-i 或 -interactive：覆盖之前询问的用户
-r：递归处理，将指定目录下的文件与之目录一并处理，若源文件或目录的形态，不属于目录或符号链接，则一律视为普通文件
-R或-recursive：递归处理，将 指定文件与子目录一并处理。


7.mv指令（剪切，重命名）
语法：mv [选项] 源文件或目录 目标文件或目录
功能：
1）当第二个参数是文件时，mv命令完成文件重命名，此时源文件只能有一个（也可以是源文件目录名），他将所给的源文件或目录重命名为给定的目标文件名。
2）当第二个参数是已存在的目录名称时，源文件或目录参数可以是多个，mv命令将各个参数指定的源文件均移至目标文件中。
说明：mv 时move的缩写，可以用来备份文件或目录的重命名。
mv  源文件名  已经存在的路径：把源文件拷贝到该路径下
mv  原文件名   新文件名（不存在的文件）：把源文件重命名     需要自己实验验证对错

8.rmdir指令和rm 指令
rmdir指令和mkdir指令相对应，mkdir是建立一个目录，而rmdir是删除命令。
语法：rmdir [-p]  目录名
适用对象：具有当前目录操作权限的使用者
功能：删除空目录
常用选项：
-p：当子目录被删除后如果父目录也是空目录，就把父目录一起删除 。

rm命令：可以同时删除文件或目录
语法：rm  [选项]  文件名/目录名
适用对象：所有使用者
功能：删除文件或目录
常用选项：
-f ：force，强制删除文件或者目录，即使文件制度（写保护），也可以直接删除
-r：删除目录及其下所有文件
-i：删除前逐一询问确认

rm  -r  文件名：删除文件
rm   -f  文件名、目录名：强制删除文件 或目录

9.man指令
man指令是查找对应的库函数，以及通过man手册获取帮助。
常用选项
-k：根据关键字搜索联机帮助
num ：只在第num章节查找对应信息
-a：将所有信息都显示出来，举例：man  scanf 她会缺省总第一章开始搜索，找到就停止。当按q退出，他会继续向后边搜索，直到所有章节都搜索完毕。

手册：
1：它是普通命令
2：它是系统调用，如open，write之类的（通过这个可以很方便的查到这个函数加什么头文件）
3：是库函数，如printf，fread
4：是特殊文件，也就是/dev下的各种设备文件。
5：指文件的格式，比如passwd，就会说明这个文件中的各个字段的含义
6：是给游戏留的，由各个游戏自己定义
7：时附件好友一些变量，比如environ这种全局变量
8：是系统管理用的文件命令，这些命令只能由root使用，如ifconfig


10.cat/tac指令
语法：cat   [选项]  [文件]
功能：查看目标文件的内容
常用选项：

-b ：对费控输出行编号
-n：对输出的所有行进行编号
-s：不输出多行空行

补充：

cd  -R  目录名  新目录名  ：把文件拷贝到当前目录    cd  -R  code  /code
echo 字符串  ：把字符串显示到屏幕上
>：重定向

举例：
echo  “hello” > file  ：把hello输出到文件file中
cat  file：显示文件中的内容

11.more指令（只能向下翻）
语法： more  [选项] [文件]
功能：与cat类似显示文件内容，但是more指令当屏幕写满后就不向下显示内容了，如果想要继续向下看内容就按“向下的箭头”
常用选项：

-n  :设置默认输出的行数，n代表几行
more -2：输出两行

q：退出more

cat和more的区别：
cat输出文件的全部内容，more默认输出满屏

补充：
1.>：重定向（输出重定向，把原来的内容清空，再把新的数据输入到该文件中）
例子：ls  /> name  :把根目录中的内容定下输出到name中。
2.>>：追加重定向（不清除原来文件的内容，直接将内容增加到原文件）
3.|：管道连接两条命，将左边命令的输入，作为右边命令的输出。
例子：ls  -l  /  |  more  :把根目录下的内容以more的形式输出

12.less指令（不仅可以向下翻，也可以向上翻）
语法：less [参数]  文件名
功能：less与more类似，但是less可以随意浏览文件，而more仅能向下移动，不能向上移动，我而且less查看文件时不会加载整个文件。
常用选项：

-i：搜索是忽略大小写
-N：显示每行的行号
/字符串：向下搜索”字符串”的功能
？字符串：向上搜索”字符串”的功能
n：重复前一个搜索（与/或？有关）
N：反复重复前一个搜索（与/或？有关）
q：quit退出

说明：
1.less工具也是对文件或其他输出行进行分页显示的工具，应该说是Linux正统查看文件内容的工具，功能强大
2：less比more更加灵活，因为more只能向下翻，而less向下向上翻都可以。
3.less有更多搜索功能，不仅可以向下搜索，也可以向上搜索

14.tail指令
语法： tail  [参数]  文件名
功能：由于显示文件末尾的内容，不指定文件时，作为输入信息进行处理，常用来查看日志文件。
常用选项：

-f ：循环读取
-n：显示行数
说明：tail指令从指定开始的文件写到标准输出，是一样tail指令的-f 选项可以方便的查阅正在改变的日志文件，tail -f 文件名会把文件里的最尾部的内容显示在屏幕上，，并且不断 刷新，使看到最新的文件内容。


15.与时间相关的指令
date显示
date指定格式显示时间，date+%Y:%m:%d
date用法：date [选项]  [格式]

1）.在显示方面，使用者可以设定欲显示的格式，考个试后边加一个结束标记，其中常用的标记如下、

%H：小时（00...23）
%M：分钟（00...59）
%S：秒（00...60），标注这里有问题，秒是60还是61
%X：相当于%H%:M:%S
%d : 日（01...31）
%m：月份（01..12）
%Y：完整年份（0000...9999）
%F：相当于 %Y-%m-%d

举例：
date +%Y:%m：%d： 2019:1:18
date +%Y/%m/%d： 2019/1/18
date +%Y:%m：%d-%H:%M:%S : 2019:1:18-15:19:23
date +%Y:%m：%d-%H:%M:%S -d  @时间戳打印成对于的时间

2）.在设置时间方面

date -s   :设置当前时间，只有root权限才能设置，其他的只能查看
date -s  20100502：设置成20100502，这样会把具体时间设置成00:00:00
date  -s 01:01:01：设置具体时间，不会对日期做更改
date -s "01:01:01 2010-05-02 "：设置全部时间
date -s "01:01:01 20100502 ":设置全部时间
date -s " 2010-05-02 01:01:01":设置全部时间
date -s "20100502  01:01:01  ":设置全部时间

3）.时间戳
时间->时间戳：date+%s

时间戳是从1970.01.01的午夜开始所经历的秒数，不考虑闰秒
特点： 时间戳是一直不断增大的，而且不会重复，所以这个常用来查找错误
